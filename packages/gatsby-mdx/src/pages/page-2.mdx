# Code Snippet Demo

## Javascript

```js theme=github showLineNumbers
function greet({ name = 'Rauno' } = {}) {
	console.log(`Hi ${name}!`);
}

greet(); // Hi Rauno!
greet({ name: 'Larry' }); // Hi Larry!
```

## Kotlin

```kotlin showLineNumbers
fun callbackIfTrue(condition: Boolean, callback: () -> Unit) {
    if (condition) {
        callback()
    }
}

callbackIfTrue(someBoolean) {
    print("Condition was true")
}
```

## Ruby

```ruby
items = Model
  .select(:id)
  .select([:id, :name])
  .group(:name)   # GROUP BY name
  .group('name AS grouped_name, age')
  .having('SUM(price) > 30')  # needs to be chained with .group
  .includes(:user)
  .includes(user: [:articles])
  .references(:posts)
  # aka: .where("posts.name = 'foo'").references(:posts)
```
